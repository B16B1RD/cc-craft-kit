## 1. 背景と目的

### 背景

v0.1.2 のリリースにあたり、最新仕様をドキュメント類に反映する。また、バージョンを v0.1.1 から v0.1.2 にアップデートするので、その点の見直しも進める。

### 目的

v0.1.2 リリースに向けて、以下の目標を達成する:

1. 古いプロジェクト名（"Takumi"）を "cc-craft-kit" に統一
2. 統合されたコマンド体系（42個→24個）をドキュメントに正確に反映
3. 廃止されたフェーズ（tasks）の記載を更新
4. package.json のバージョンを v0.1.1 から v0.1.2 に更新
5. CHANGELOG.md に v0.1.2 の変更内容を追記

## 7. 設計詳細

### 7.1 アーキテクチャ設計

#### 全体構成

本タスクはドキュメント更新のみのため、コード変更なし。対象ファイルは以下の構造:

```
cc-craft-kit/
├── README.md                          # メインドキュメント（コマンド一覧再構成）
├── CLAUDE.md                          # Claude Code 統合仕様（確認のみ）
├── CHANGELOG.md                       # バージョン履歴（v0.1.2追加）
├── package.json                       # バージョン情報（v0.1.2更新）
└── docs/
    ├── GITHUB_PROJECTS.md             # プロジェクト名統一（16箇所）
    ├── SUBAGENTS_AND_SKILLS_GUIDE.md  # プロジェクト名統一（3箇所）
    ├── ARCHITECTURE.md                # バージョン・更新日修正
    └── QUICK_START.md                 # 確認のみ
```

#### 設計方針

1. **プロジェクト名統一**: "Takumi" → "cc-craft-kit" の一括置換
2. **コマンド一覧再構成**: 削除済みコマンドの除去、統合コマンドの追加
3. **フェーズフロー更新**: 4フェーズモデル（tasks 非推奨化）への統一
4. **バージョン更新**: v0.1.1 → v0.1.2 の一括更新

#### 処理フロー詳細

```
Phase 1: プロジェクト名統一
  ├── docs/GITHUB_PROJECTS.md（16箇所）
  └── docs/SUBAGENTS_AND_SKILLS_GUIDE.md（3箇所）
           ↓
Phase 2: README.md コマンド一覧再構成
  ├── 削除: 18個の未実装コマンド
  ├── 追加: 統合コマンド説明
  └── カテゴリ構成の整理
           ↓
Phase 3: フェーズフロー更新
  ├── README.md フェーズ説明
  └── docs/ARCHITECTURE.md フェーズ定義
           ↓
Phase 4: バージョン情報更新
  ├── package.json（v0.1.2）
  ├── CHANGELOG.md（v0.1.2 セクション追加）
  └── docs/ARCHITECTURE.md（バージョン・更新日）
           ↓
Phase 5: 最終検証
  └── リンク・用語の整合性確認
```
