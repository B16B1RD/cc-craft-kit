## 1. 背景と目的

### 背景

v0.1.2 のリリースにあたり、最新仕様をドキュメント類に反映する。また、バージョンを v0.1.1 から v0.1.2 にアップデートするので、その点の見直しも進める。

### 目的

v0.1.2 リリースに向けて、以下の目標を達成する:

1. 古いプロジェクト名（"Takumi"）を "cc-craft-kit" に統一
2. 統合されたコマンド体系（42個→24個）をドキュメントに正確に反映
3. 廃止されたフェーズ（tasks）の記載を更新
4. package.json のバージョンを v0.1.1 から v0.1.2 に更新
5. CHANGELOG.md に v0.1.2 の変更内容を追記

---

## 2. 対象ユーザー

- cc-craft-kit を使用する開発者
- cc-craft-kit を導入検討中のユーザー
- cc-craft-kit のメンテナー

## 3. 受け入れ基準

### 必須要件

- [ ] 全ドキュメントで "Takumi" → "cc-craft-kit" に統一されている
- [ ] README.md のコマンド一覧が実装済みの24コマンドと一致している
- [ ] package.json のバージョンが v0.1.2 に更新されている
- [ ] CHANGELOG.md に v0.1.2 セクションが追加されている

### 機能要件

- [ ] README.md の「全コマンド一覧」セクションが再構成されている
- [ ] フェーズ説明から tasks が削除され、design → implementation の流れが明記されている
- [ ] 統合コマンド（/cft:task, /cft:knowledge, /cft:custom-tools, /cft:quality, /cft:sync）の説明が追加されている

- [ ] docs/SUBAGENTS_AND_SKILLS_GUIDE.md の古いプロジェクト名が修正されている

### 非機能要件

- [ ] ドキュメント内のリンクが全て有効である

- [ ] 用語・表記が統一されている
