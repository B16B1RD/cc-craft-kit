# v0.1.9 リリース準備


## 1. 背景と目的

### 背景

最新の develop ブランチの内容を v0.1.9 としてリリースします。リリース番号に関わる部分を調査し、修正する作業です。

### 目的

cc-craft-kit v0.1.9 を正式リリースするため、バージョン番号の更新、CHANGELOG の記載、ドキュメント更新を行い、GitHub Release を通じて配布可能な状態にする。

## 2. 対象ユーザー

## 4. 制約条件

- Semantic Versioning (MAJOR.MINOR.PATCH) に準拠
- Keep a Changelog 形式でログを記述
- GitHub Flow + develop ブランチ戦略に従い、develop → main へ PR 経由でマージ
- Git タグは main ブランチで付与（release.yml が自動トリガー）
---

## 5. 依存関係

- package.json: バージョン番号の源泉
- CHANGELOG.md: 変更履歴
- docs/ARCHITECTURE.md: ドキュメント内のバージョン表記
- .github/workflows/release.yml: 自動リリースワークフロー
- scripts/install.sh: GitHub Pages へデプロイされるインストールスクリプト
