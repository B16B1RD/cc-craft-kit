## 1. 背景と目的

### 背景

cc-craft-kit プロジェクトの docs/ ディレクトリには、プロジェクトのリブランディング前に使用していた商標関連ファイルおよび旧マイグレーションガイドが残存しています。これらのファイルはプロジェクトの現在の状態と整合性がなく、ドキュメント構造を複雑化させています。

### 目的

プロジェクトドキュメントの整理と保守性向上のため、以下の不要ファイルおよびディレクトリを削除します。

- `docs/trademark-evidence/` ディレクトリ（商標証拠資料）
- `docs/MIGRATION_GUIDE.md`（旧マイグレーションガイド）
- `docs/trademark-research-report.md`（商標調査レポート）
- `docs/trademark-template.md`（商標関連テンプレート）
- `MIGRATION.md`（不完全な移行ガイド、実質的に意味をなしていない）
- `.takumi/` ディレクトリ（旧プロジェクト名時代のディレクトリ、現在は `.cc-craft-kit/` を使用）

## 6. 参考情報

### 削除理由

- `trademark-evidence/`: プロジェクトのリブランディング前の商標調査資料で、現在は不要
- `MIGRATION_GUIDE.md`: 旧プロジェクト構造からの移行ガイドで、移行完了後は不要
- `trademark-research-report.md`: 旧商標調査レポートで、リブランディング後は不要
- `trademark-template.md`: 商標関連テンプレートで、現在の運用では不要
- `MIGRATION.md`: 不完全な移行ガイド（cc-craft-kit → cc-craft-kit という同じ名前への移行で、実質的に意味をなしていない）
- `.takumi/`: 旧プロジェクト名時代のディレクトリで、現在は `.cc-craft-kit/` を使用しているため不要

### 参照チェックコマンド

削除前に、削除対象ファイルを参照しているファイルがないか確認してください。

```bash
# 削除対象ファイルへの参照を検索
grep -r "trademark-evidence\|MIGRATION_GUIDE\|trademark-research-report\|trademark-template" \
  --include="*.md" \
  --include="*.ts" \
  --include="*.js" \
  --include="*.json" \
  .
```

**確認済みの参照:**
- `MIGRATION.md` に `docs/trademark-research-report.md` への参照が 3 箇所存在（226 行目、257 行目、265 行目）
- `.textlintignore` に `MIGRATION.md`, `docs/trademark-research-report.md`, `.takumi/` の記載がある（3 行目、18 行目、19 行目）
- `.gitignore` に `.takumi/` の記載がある（33 行目、34 行目）
- その他のスクリプトファイル（`src/scripts/migrate-takumi-to-cc-craft-kit.ts` など）にも参照があるが、これらは削除または修正が必要

### 削除コマンド例

```bash
# 1. 参照を削除または修正
# .textlintignore から MIGRATION.md, trademark-research-report.md, .takumi/ の行を削除
# .gitignore から .takumi/ の行を削除

# 2. ディレクトリ削除
git rm -r docs/trademark-evidence/
rm -rf .takumi/  # .gitignore で除外されているため git rm 不要

# 3. ファイル削除
git rm docs/MIGRATION_GUIDE.md \
       docs/trademark-research-report.md \
       docs/trademark-template.md \
       MIGRATION.md
```
