# docs/archive/ や docs/MIGRATION_FROM_MCP.md の不要ファイルの削除


## 1. 背景と目的

### 背景

cc-craft-kit プロジェクトのドキュメント構成に、以下の不要ファイルが残っています。

1. **`docs/archive/` ディレクトリ**
   - `docs/archive/NEW_ARCHITECTURE_DESIGN.md`
   - `docs/archive/FINAL_ARCHITECTURE_DESIGN.md`
   - これらはアーキテクチャ設計の初期検討資料であり、現在は `docs/ARCHITECTURE.md` に統合済み

2. **`docs/MIGRATION_FROM_MCP.md`**
   - MCP サーバーからカスタムスラッシュコマンドへの移行ガイド
   - プロジェクトがすでに移行完了しており、今後は不要

これらのファイルを参照しているドキュメントが存在するため、削除前にリンクの修正が必要です。

### 目的

1. 不要なドキュメントファイルを削除してリポジトリを整理
2. 他ドキュメント内の参照リンクを修正または削除
3. ドキュメント構成を明確化し、保守性を向上
---

## 2. 対象ユーザー

- **開発者**: cc-craft-kit プロジェクトの開発者
- **コントリビューター**: ドキュメントを参照する外部貢献者
- **新規利用者**: ドキュメントを読んでプロジェクトを理解するユーザー

## 4. 制約条件

- Git 履歴からファイルを完全に削除する必要はない（通常の `git rm` でよい）
- 削除後も、ドキュメント全体の構成が保たれていること
- `docs/ARCHITECTURE.md` に統合されているため、アーカイブファイルの内容は保存不要
